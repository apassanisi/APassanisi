### Hi, I'm Andy.

I'm a frontend engineer with 10 years of experience across agencies, product teams, and my own studio. Most of my recent work is in **Vue 3** and **TypeScript** — component libraries, design systems, and the kind of dense, data-rich interfaces most frameworks avoid.

I run **[Amulet Laboratories](https://amuletlabs.org)** out of Eureka, California. Through the studio I offer two kinds of work — **building** new sites and software for small businesses in Northern California (hotels, coffee shops, therapy practices, dispensaries, and other local operators), and **auditing** existing Vue 3 codebases, accessibility surfaces, and design systems for product teams that already have one.

### Currently building

- **[Rig](https://github.com/Amulet-Laboratories/Rig)** — Headless Vue 3 component library. 149 components across 12 packages, completely unstyled, accessible by default.
- **Hex** — Companion 27-theme CSS framework that targets Rig's `data-rig-*` attributes. One import swaps an entire visual identity.
- **Obelisk** — Tauri 2 + Vue 3 desktop app. A structured second brain over a markdown vault, with 31 feature modules and AI-tool integration.
